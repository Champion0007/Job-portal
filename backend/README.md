# Job Portal Backend

This folder contains a starter Express + MongoDB backend for the Job Portal.

Quick start

1. Copy `.env.example` to `.env` and fill in values.
2. Install dependencies:

```powershell
cd backend
npm install
```

3. Start in development:

```powershell
npm run dev
```

APIs are exposed under `/api/*`. This repo includes starter models, routes and services for authentication, jobs, applications and payments.
